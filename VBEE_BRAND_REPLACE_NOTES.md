# Vbee Brand Replacement

Đã cập nhật toàn bộ nhận diện trong project sang Vbee.

## Đã thay đổi

- Thêm logo chính `vbee-logo.png` từ hình bạn gửi.
- Thêm icon `vbee-icon.png`.
- Thay favicon bằng icon Vbee.
- Cập nhật toàn bộ chữ thương hiệu trong source code và tài liệu sang `Vbee`.
- Tài khoản demo mới trong backend: `demo@vbee.local` / `123456`.

## Chạy project

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Mở: `http://localhost:3000`
